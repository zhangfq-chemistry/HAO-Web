# HydrogenAtomOrbitalViewer 使用手册

本文档根据当前项目源码、Qt UI 文件、资源文件和原始 `readme` 整理而成，可作为软件使用说明、课程材料和论文/文章撰写的基础资料。项目生成的 `build/`、`.qtc_clangd/`、`Makefile`、`qrc_application.cpp` 等文件可由 Qt/qmake 或 IDE 重新生成，日常使用和二次开发主要关注 `src.pro`、`*.cpp`、`*.h`、`UI/`、`images/`、`translations/`。

## 1. 软件概述

`HAO` 是一个基于 Qt Widgets、Qt Charts、Qt Data Visualization 和 VTK 的桌面程序，用于显示氢原子轨道和杂化轨道。程序启动后会显示启动图，主窗口分为左侧控制面板和右侧可视化区域。

软件主要功能：

- 氢原子轨道选择：支持 `1S` 到高主量子数的 `S/P/D/F/G` 轨道。
- 三维轨道可视化：显示正瓣、负瓣、网格面、光滑面、电子云、节面和裁切视图。
- 辅助视图：角向函数、径向函数、轨道投影。
- 杂化轨道可视化：支持 `SP`、`SP2`、`SP3`、`D2SP`、`DSP3`、`D2SP3` 以及单独的 `S/P/D` 原子轨道示意。
- 图片导出：可将当前活动视图保存为 PNG。
- 多语言界面：启动时按系统语言加载中文或英文翻译，也可通过菜单切换。

### 1.1 软件定位

该软件面向量子化学、结构化学和普通化学教学中的轨道可视化需求。它将氢原子波函数、实球谐角向函数、径向分布函数和常见杂化轨道以交互式三维图形呈现，便于观察轨道的空间取向、正负相位、节面数量、径向分布和杂化构型。

与静态教材插图相比，本软件的优势在于：

- 可交互旋转三维轨道，观察不同视角下的形状变化。
- 可切换网格、光滑面、电子云、截面和投影显示方式。
- 可调节等值面、盒子尺寸、网格点数和颜色透明度。
- 可导出 PNG 图片，用于实验报告、教学课件和文章插图。

### 1.2 适用场景

- 结构化学课程中解释氢原子轨道形状。
- 量子化学课程中演示 `n`、`l`、`m` 对轨道形状的影响。
- 分析径向节面、角向节面和轨道相位。
- 展示 `sp`、`sp2`、`sp3`、`d2sp`、`dsp3`、`d2sp3` 等杂化轨道。
- 制作轨道可视化教学图片和论文配图。

## 2. 理论基础

本节给出使用和写作时常用的理论背景。软件源码中的主要数学实现位于 `functionals.cpp`、`AO.cpp`、`ylmFunctions.cpp` 和 `mixedFunctions.cpp`。

### 2.1 氢原子波函数

氢原子轨道可表示为径向部分和角向部分的乘积：

```text
Psi(n,l,m;r,theta,phi) = R(n,l;r) * Y(l,m;theta,phi)
```

其中：

- `n` 为主量子数，决定轨道能级和径向尺度。
- `l` 为角量子数，决定轨道类型，`l=0,1,2,3,4` 分别对应 `S/P/D/F/G`。
- `m` 为磁量子数，决定轨道空间取向。
- `R(n,l;r)` 是径向函数。
- `Y(l,m;theta,phi)` 是球谐函数或其实函数组合。

软件中使用实轨道形式显示常见化学轨道，例如 `Px`、`Py`、`Pz`、`Dxy`、`Dxz`、`Dz2` 等。对 `m != 0` 的轨道，界面中的 `Cos型` 和 `Sin型` 对应不同的实球谐组合。

教学说明中应强调：完整轨道形状来自径向部分和角向部分的乘积。角向函数决定方向性和角向节面，径向函数决定轨道随半径变化的振荡、径向节面和概率分布范围。若只展示角向函数，学生可能误以为 p、d、f 轨道的三维形状完全由角向部分决定；若只展示径向分布，又会失去轨道方向性。本软件把 `原子轨道`、`角向函数`、`径向函数` 和 `轨道投影` 分开显示，适合用来说明这四类图像各自回答的问题不同。

### 2.2 概率密度和电子云

电子在空间中出现的概率密度与波函数模平方有关：

```text
rho(r,theta,phi) = |Psi(n,l,m;r,theta,phi)|^2
```

软件中的等值面显示对应 `Psi` 或概率相关函数达到某一阈值时形成的空间曲面；电子云显示则用空间点分布表现电子出现概率的高低。

在文章中可这样表述：

> 通过调节等值面阈值，可以观察同一轨道在不同概率密度范围下的空间轮廓。电子云模式则更直观地表现了电子概率密度的空间分布。

需要注意，等值面、点云、截面和径向分布函数不是互相替代的图像，而是互相补充的表示。等值面便于观察轨道边界、相位和对称性，但容易让初学者误以为电子只存在于曲面附近；电子云更接近概率分布的直观图像，但难以准确显示节面和相位；截面适合观察内部节点；RDF 适合解释电子在不同半径附近出现的概率。文章中可把这四类图像作为同一轨道的“多视角证据”。

### 2.3 径向函数、平方和径向分布函数

软件提供三类径向显示：

- `R`：径向波函数 `R(r)`。
- `R2`：径向函数平方，反映径向函数幅值平方。
- `RDF`：径向分布函数，通常用于描述电子在半径 `r` 附近球壳中出现的概率分布。

对文章写作而言，`RDF` 更适合解释“电子离原子核多远处最可能出现”；`R` 和 `R2` 更适合讨论波函数符号、节点和函数形式。

建议在教学活动中把 `R`、`R2`、`RDF` 与三维轨道联合使用。例如先观察 `3Pz` 的三维双瓣结构，再切换到径向函数窗口比较 `R` 的符号变化、`R2` 的非负性和 `RDF` 的峰位。这样可以说明：轨道图像中的径向节点不是一个孤立的绘图效果，而是径向波函数取零导致的空间球形节点。

### 2.4 节面

节面是波函数取零值的区域。氢原子轨道常见节面包括：

- 径向节面：由径向函数为零产生，通常表现为球形节点。
- 极向节面：由角向函数的极角部分产生。
- 经向节面：由角向函数的方位角部分产生。

软件会根据当前轨道的实际节面数量自动启用或禁用相应选项。文章中可结合节面数量说明轨道形状与量子数之间的关系。

### 2.5 杂化轨道

杂化轨道是若干原子轨道的线性组合，常用于解释分子构型和成键方向。例如：

- `SP`：常用于线形构型。
- `SP2`：常用于平面三角形构型。
- `SP3`：常用于四面体构型。
- `D2SP`、`DSP3`、`D2SP3`：用于展示含 d 轨道参与的扩展杂化模型。

软件提供角向波函数和示意图两种模式，可分别用于理论形状展示和结构构型示意。

### 2.6 实轨道、复轨道和相位解释

量子力学教材中常用的球谐函数可以是复函数形式，而化学教学中常见的 `Px`、`Py`、`Dxy`、`Dxz` 等轨道通常是由复球谐函数线性组合得到的实轨道。本软件主要显示实轨道，并用正瓣、负瓣颜色表示波函数符号或相位差异。这种表示方式非常适合解释成键方向、节面和杂化轨道，但它并不等同于完整复波函数相位可视化。

写文章时建议这样处理：在理论部分说明软件采用实球谐组合来匹配化学教学中常见的轨道命名；在局限部分说明当前版本没有提供复轨道相位、复值分子轨道或相位色轮等高级表示。这样既能保持教学叙述清楚，也能避免把本软件描述成通用波函数分析软件。

### 2.7 节点数和量子数的对应关系

氢原子轨道的节点规律可作为文章和课堂讲解的一条主线：

- 角向节点总数为 `l`。
- 径向节点数为 `n-l-1`。
- 总节点数为 `n-1`。

例如 `2P` 轨道有 1 个角向节点、0 个径向节点；`3P` 轨道有 1 个角向节点、1 个径向节点；`3D` 轨道有 2 个角向节点、0 个径向节点。使用本软件时，可通过勾选 `径向`、`极向`、`经向` 节面，并结合截面或裁切视图，让节点数从公式变成可观察的空间结构。

## 3. 快速开始

### 3.1 显示第一个原子轨道

1. 启动程序。
2. 保持默认轨道 `1S`。
3. 点击左侧 `显示`。
4. 在右侧观察 `原子轨道`、`角向函数`、`径向函数` 和 `轨道投影` 四个窗口。
5. 点击工具栏 `储存图片(PNG)` 导出当前活动窗口。

### 3.2 显示 `3Dz2` 轨道

1. 将 `n` 设为 `3`。
2. 在轨道下拉框选择 `3Dz2`。
3. 点击 `显示`。
4. 勾选 `极向` 或 `径向` 节面，观察节面与轨道形状的关系。
5. 勾选 `xz截面` 或 `yz截面`，观察轨道内部结构。

### 3.3 显示 `SP3` 杂化轨道

1. 切换到 `杂化轨道` 页签。
2. 在杂化轨道下拉框选择 `SP3`。
3. 选择 `角度波函` 或 `示意图`。
4. 使用 `正瓣`、`负瓣`、`网格`、`光滑` 调整显示方式。
5. 使用旋转按钮调整观察角度并导出图片。

## 6. 氢原子轨道使用流程

### 6.1 选择轨道

在左侧 `氢原子轨道` 面板的 `波函数` 分组中操作：

1. 选择主量子数 `n`，源码中初始最大值为 `6`。
2. 根据 `n` 自动限制角量子数 `l` 的范围为 `0` 到 `n-1`。
3. 根据 `l` 自动限制磁量子数 `m` 的范围为 `-l` 到 `l`。
4. 也可直接从轨道下拉框选择轨道名称，例如 `1S`、`2Px`、`3Dz2`、`5Gxyz2`。
5. 对 `m != 0` 的轨道，可以选择 `Cos型` 或 `Sin型`。程序内部用正负 `m` 区分两种实球谐组合。
6. 点击 `显示`，程序会计算轨道并刷新全部相关视图。

轨道命名规则：

- `S`: `m = 0`
- `Pz`: `m = 0`
- `Px`: `m > 0`
- `Py`: `m < 0`
- `Dz2`, `Dxz`, `Dyz`, `Dxy`, `Dx2-y2`
- `Fz3`, `Fxz2`, `Fyz2`, `Fz(x2-y2)`, `Fxyz`, `Fx(x2-3y2)`, `Fy(3x2-y2)`
- `Gz4`, `Gxz3`, `Gyz3`, `G(x2-y2)z2`, `Gxyz2`, `Gx3z`, `Gy3z`, `Gxy(x2-y2)`, `G(x4+y4)`

### 6.2 视图窗口

点击 `显示` 后，程序会刷新：

- `原子轨道`：三维等值面或电子云。
- `角向函数`：角向部分轮廓图。
- `径向函数`：径向函数、径向概率密度或径向分布函数。
- `轨道投影`：轨道投影视图。

### 6.3 轨道显示选项

`轨道显示` 分组包含：

- `正瓣`：显示或隐藏正相位轨道瓣。
- `负瓣`：显示或隐藏负相位轨道瓣。
- `网格`：以网格方式显示等值面。
- `光滑`：以光滑表面显示等值面。
- `xy截面`、`xz截面`、`yz截面`：只显示对应平面截面。
- `1/8角`、`1/4角`、`3/4角`：显示不同裁切区域，便于观察内部结构。
- `投影`：控制轨道投影视图中的半轨道/投影显示状态。
- `配色I`：连续色图方式显示原子轨道表面。
- `配色II`：在不同颜色方案之间切换。
- `颜色`：打开原子轨道外观设置对话框。
- `坐标轴`：打开坐标轴属性设置对话框。

注意：

- `网格` 和 `光滑` 会互斥切换。
- 打开 `电子云` 后会隐藏等值面显示。
- 打开连续色图 `配色I` 时，网格显示会被关闭。

### 6.4 节面显示

`节面` 分组包含：

- `径向`：显示径向节面。
- `极向`：显示极向节面。
- `经向`：显示经向节面。

点击 `显示` 后，程序会根据当前轨道实际节面数量自动禁用不存在的节面选项。

### 6.5 盒子和网格参数

`盒子` 分组包含：

- `总概率`：当前盒子范围覆盖的总概率，界面中默认只读。
- `间隔`：格点间隔，当前槽函数直接返回，通常由程序计算。
- `尺寸`：盒子尺寸，修改后点击 `重画` 生效。
- `格点数`：三维网格点数，数值越大越精细但越慢。
- `等值面`：轨道等值面阈值。
- `显示盒子`：显示或隐藏边界盒。
- `3D网格`：显示或隐藏三维网格。
- `预设值`：恢复当前轨道计算出的默认盒子、等值面、格点数、间隔和概率。
- `重画`：用当前盒子尺寸、格点数、等值面重新计算并刷新。

建议：

- 想要更平滑的等值面，可增加 `格点数`，但计算时间会明显增加。
- 如果轨道显示太小或被截断，增大 `尺寸` 后点击 `重画`。
- 如果轨道表面太厚或太薄，调整 `等值面` 后点击 `重画`。

### 6.6 径向函数显示

`径向` 分组包含：

- `R`：显示径向波函数 `R(r)`。
- `R2`：显示径向函数平方。
- `RDF`：显示径向分布函数，默认开启。
- `曲线`：显示曲线。
- `横截`：显示横截效果。
- `侧视`：显示侧视二维效果。
- `坐标`：显示坐标轴。

`R`、`R2`、`RDF` 在当前实现中互斥，勾选一个会取消另外两个。

### 6.7 原子轨道颜色设置

点击 `轨道显示` 中的 `颜色` 打开 `OrbitalProperty` 对话框，可设置：

- 正瓣颜色和透明度。
- 负瓣颜色和透明度。
- 径向节面颜色和透明度。
- 极向节面颜色和透明度。
- 经向节面颜色和透明度。
- 背景颜色。
- 轨道线宽和节面厚度相关参数。

按钮含义：

- `确定`：应用设置并关闭对话框。
- `取消`：关闭对话框。
- `存盘`、`载入`：UI 中存在按钮，但当前源码中未实现完整保存/载入逻辑。

### 6.8 坐标轴设置

点击 `坐标轴` 打开 `AxisProperty` 对话框，可设置：

- X、Y、Z 轴颜色。
- X、Y、Z 轴长度。
- `应用`：实时应用但不关闭。
- `确定`：应用并关闭。
- `取消`：恢复进入对话框前的坐标轴设置。

### 6.9 原子轨道面板控件对照表

| 控件 | 作用 | 备注 |
| --- | --- | --- |
| `n` | 设置主量子数 | 当前源码初始最大值为 `6`，列表中保留到 `7` 的分支。 |
| `l` | 设置角量子数 | 范围随 `n` 自动变化。 |
| `|m|` | 设置磁量子数绝对值 | 与 `Cos型`、`Sin型` 配合确定实轨道方向。 |
| `Cos型` | 选择余弦型实球谐组合 | 仅 `m != 0` 时启用。 |
| `Sin型` | 选择正弦型实球谐组合 | 与 `Cos型` 互斥。 |
| `电子云` | 显示电子云点 | 会隐藏等值面。 |
| `径向/极向/经向` | 显示对应节面 | 不存在的节面会被自动禁用。 |
| `正瓣/负瓣` | 控制轨道相位瓣显示 | 适合制作相位对比图。 |
| `网格/光滑` | 控制表面渲染模式 | 二者互斥。 |
| `xy/xz/yz截面` | 显示平面截面 | 适合展示轨道内部节点。 |
| `1/8角/1/4角/3/4角` | 显示空间裁切视图 | 适合观察复杂高阶轨道。 |
| `配色I/配色II` | 切换配色方式 | `配色I` 对应连续色图表面。 |
| `尺寸` | 设置计算盒子大小 | 修改后点击 `重画`。 |
| `格点数` | 设置三维采样精度 | 越大越精细，计算越慢。 |
| `等值面` | 设置等值面阈值 | 影响轨道表面大小。 |
| `重画` | 重新计算当前轨道数据 | 修改尺寸、格点数、等值面后使用。 |

## 7. 杂化轨道使用流程

### 7.1 选择杂化轨道

进入 `杂化轨道` 页签，在左侧下拉框中选择：

- `None`
- `SP`
- `SP2`
- `SP3`
- `D2SP`
- `DSP3`
- `D2SP3`
- `S`
- `Px`
- `Py`
- `Pz`
- `Dxy`
- `Dyz`
- `Dxz`
- `Dx2-y2`
- `Dz2`

选择后程序会：

1. 将轨道加入左侧树列表。
2. 设置当前杂化轨道名称。
3. 生成角向数据。
4. 生成示意模型。
5. 刷新杂化轨道三维视图。

### 7.2 杂化轨道显示选项

`轨道` 分组包含：

- `正瓣`：显示或隐藏正相位瓣。
- `负瓣`：显示或隐藏负相位瓣。
- `网格`：网格方式显示。
- `光滑`：光滑表面显示。
- `角度波函`：以角向波函数方式显示。
- `示意图`：以示意模型方式显示。
- `格子`：显示或隐藏示意图外框/轮廓。
- `外加S`：显示外加 S 轨道。
- `外加P`：显示外加 P 轨道。
- `外加模式`：外加轨道使用网格模式。
- `颜色`：打开杂化轨道外观对话框。
- `坐标轴`：打开坐标轴属性对话框。

`示意图` 与 `角度波函` 互斥。`外加S` 与 `外加P` 互斥。

### 7.3 盒子类型

部分杂化轨道或单独轨道显示会启用 `盒子类型` 下拉框，包含：

- `四面体`
- `立方体`
- `八面体`
- `四边形`

切换后程序会刷新杂化轨道视图。

### 7.4 杂化轨道树列表

杂化轨道页左侧树控件由 `OBTree` 实现。选择新轨道时会加入树中，树项可用于控制各轨道显示状态。

行为：

- 勾选树项：该轨道在杂化视图中可见。
- 取消勾选树项：该轨道被加入不可见列表，不再渲染。

### 7.5 杂化轨道颜色设置

点击杂化轨道面板中的 `颜色` 打开 `HybridizedOrbitalProperty` 对话框，可设置：

- 正瓣颜色和透明度。
- 负瓣颜色和透明度。
- 外加正瓣颜色和透明度。
- 外加负瓣颜色和透明度。
- 线条颜色。
- 线宽。
- `默认`：恢复默认杂化轨道配色和线宽。

### 7.6 杂化轨道面板控件对照表

| 控件 | 作用 | 备注 |
| --- | --- | --- |
| 杂化轨道下拉框 | 选择 `SP`、`SP2`、`SP3` 等轨道 | 选择后立即生成并刷新。 |
| `角度波函` | 以角向波函数模式显示 | 与 `示意图` 互斥。 |
| `示意图` | 以构型示意模式显示 | 适合教学图片。 |

## 8. 旋转和视角控制

杂化轨道面板提供 `旋转` 分组：

- `旋转角度`：每次点击按钮旋转的角度，默认 `10`。
- `顺时针`
- `逆时针`
- `上`
- `下`
- `左`
- `右`

这些按钮当前源码主要作用于杂化轨道视图 `HybridizeObitialViewer`。顶部工具栏中的 `XOY`、`YOZ`、`XOZ` 会同时调整原子轨道、角向函数、径向函数、投影和杂化轨道的视角。

## 9. 保存 PNG

使用菜单或工具栏 `储存图片(PNG)`：

1. 如果当前在 `杂化轨道` 页签，程序保存杂化轨道视图。
2. 如果当前在 `氢原子轨道` 页签，程序根据当前活动 MDI 子窗口保存：
   - `原子轨道`
   - `轨道投影`
   - `角向函数`
   - `径向函数`
3. 选择保存位置和文件名。
4. 如果文件名不以 `.png` 结尾，程序自动追加 `.png`。

如果没有活动轨道窗口，程序会提示 `No Actived Orbital!`。

## 12. 常见问题

### 12.1 编译时找不到 VTK 或 QtColorWidgets

检查 `src.pro` 中路径是否和本机一致。Windows 示例路径是作者本机路径，需要改成本机安装路径。QtColorWidgets 需要先安装，并确保库名和 `LIBS += -lQtColorWidgets2` 匹配。

### 12.2 编译时提示 `vtkActor` 没有某些 ID 方法

说明 VTK 没有按原始 `readme` 修改并重编译。按第 4.2 节补丁说明处理。

### 12.3 点击部分选项没有明显变化

部分 UI 控件在源码中保留但功能未完全实现，例如：

- `lambertW` 复选框被禁用。
- `All` 径向图选项槽函数为空。
- 原子轨道属性中的 `存盘/载入` 按钮没有完整逻辑。
- 盒子 `间隔` 的值变更槽函数直接 `return`，通常由程序计算。

### 12.4 中文 UI 出现乱码

部分 `.ui` 文件中可见历史编码遗留文本，但 `MainWindow::switchLanguage()` 会在运行时重新设置主界面大部分中文/英文文本。如果对话框中仍乱码，需要用 Qt Designer 以 UTF-8 重新保存对应 `.ui`，或在构造函数中像主窗口一样运行时设置文本。

### 12.5 PNG 保存为空或视图没有刷新

先确认：

- 已点击 `显示` 或已选择杂化轨道。
- 当前要保存的 MDI 子窗口处于活动状态。
- GPU/OpenGL/VTK 渲染窗口工作正常。
- 修改盒子或等值面后已点击 `重画`。

### 12.6 命令行 `qmake` 报 MSVC 或工具链错误

如果命令行中默认找到的不是 Qt 6.10.1 MinGW 版本的 `qmake`，可能会出现工具链不匹配错误。建议显式运行：

```powershell
D:\Qt\6.10.1\mingw81_64\bin\qmake.exe src.pro
```

并确保后续使用对应 MinGW 工具链编译。

### 示例 4：比较 `2Pz` 和 `3Pz`

1. 先选择 `2Pz`，点击 `显示`。
2. 导出 `原子轨道`、`角向函数` 和 `径向函数/RDF` 图。
3. 再选择 `3Pz`，保持相同观察方向和相近显示参数。
4. 勾选 `径向` 节面，并使用 `xz截面` 或 `1/4角` 观察内部结构。
5. 比较两组图：两者角向形状相似，但 `3Pz` 具有额外径向节点和不同径向分布。
6. 在文章中可据此说明：完整轨道由 `R(n,l;r)` 和 `Y(l,m;theta,phi)` 共同决定。

文章引言和相关工作部分可以按以下顺序展开：

1. 轨道可视化的教学必要性。说明原子轨道和杂化轨道包含三维形状、相位、节点和概率分布等抽象概念，单靠公式和静态教材图往往难以帮助学生建立空间图像。
2. 早期图示和数学绘图方法。可概述物理模型、轮廓图、Mathematica、Gnuplot、Winplot 等方法，这些方法强调从数学表达式生成轨道图像，但对非编程用户不够直接。
3. 交互式和 Web/Notebook 可视化工具。引入 Hanson 的氢原子轨道可视化工具、Tully 等人的点描式轨道可视化，以及在线轨道探索类工具，说明交互旋转、参数调节和多种图像表示已经成为教学工具的重要方向。
4. 径向分布与角向函数的联合理解。结合 Manae 和 Hazra、Rachek 和 Wilson 等文献，强调轨道形状应由径向部分和角向部分共同解释，RDF 对理解电子距离分布尤其重要。
5. 杂化轨道和分子构型可视化。结合 Saputra 等关于三维杂化轨道教学的工作，说明从原子轨道形状过渡到分子几何构型是结构化学教学中的自然延伸。
6. 通用分子可视化软件与本软件的边界。介绍 VMD、Avogadro、Gabedit、Multiwfn 等工具的广泛能力，再说明本软件选择更聚焦的教学型设计：围绕氢原子轨道和杂化轨道提供清晰、可导图、可课堂使用的界面。
7. 新工具的贡献。最后指出本软件的主要贡献不是提出新的量子化学计算方法，而是整合实轨道、径向函数、角向函数、节面、电子云、投影和杂化轨道显示，为本科教学和文章配图提供一个本地化、交互式平台。

### 14.6 引言段落示例

可在文章初稿中改写以下段落：

> 原子轨道是量子化学和结构化学教学中的核心概念，但其数学表达式包含径向函数、角向函数、相位和节点等多层信息，学生往往难以仅凭公式或静态教材插图形成稳定的三维图像。已有研究通过 Mathematica、Gnuplot、Winplot、Web 工具、Jupyter 界面、点云模型、VR/AR 和 3D 打印等方式改进轨道可视化教学，表明交互式和多表示形式有助于学生理解轨道形状、概率密度和节点结构。然而，许多工具或偏重脚本绘图，或偏重通用分子可视化，或需要网络和特定运行环境。为此，本文基于 Qt 和 VTK 设计 HydrogenAtomOrbitalViewer，将氢原子轨道、角向函数、径向函数、投影、电子云、节面和杂化轨道整合到一个桌面图形界面中，以支持课堂演示、学生自学和教学图像生成。

### 14.7 方法段落扩展示例

> 程序将氢原子波函数表示为径向函数与实球谐角向函数的乘积。用户可通过主量子数 `n`、角量子数 `l`、磁量子数 `m` 或预设轨道名称选择目标轨道。三维轨道视图使用 VTK 渲染正负相位等值面、电子云点分布、径向/极向/经向节面、截面和裁切视图；径向窗口显示 `R`、`R2` 或 `RDF`，用于比较波函数振荡、概率密度和电子径向分布；角向函数窗口用于展示实球谐函数的方向性；投影窗口用于辅助观察三维结构在平面上的投影。对于杂化轨道，程序提供 `SP`、`SP2`、`SP3`、`D2SP`、`DSP3` 和 `D2SP3` 等模式，并允许在角向波函数和构型示意图之间切换。

### 14.8 讨论段落示例

> 与只显示轨道边界面的图像相比，本软件同时提供电子云、截面、RDF 和节面显示，有助于避免学生把轨道误解为固定边界的实体。以 `2Pz` 和 `3Pz` 为例，两者具有相同的 p 型角向特征，但 `3Pz` 由于径向函数中出现节点而呈现额外的内部结构。该对比说明，轨道图像并非仅由角向函数决定，而是径向分布和角向分布共同作用的结果。类似地，正负瓣颜色可用于说明波函数相位，而电子云和 RDF 则更适合讨论概率密度及其空间分布。多窗口联动显示使这些概念能够在同一操作流程中被观察和比较。

## 15. 已知限制

- 当前构建配置包含本机绝对路径，跨机器编译前需要修改 `src.pro`。
- 原始 `readme` 要求修改 VTK `vtkActor`，这会增加环境部署难度。
- 部分 UI 文本存在历史编码遗留问题，主窗口大部分文本会在运行时重设，但个别对话框仍可能显示乱码。
- `lambertW`、径向 `All`、属性对话框保存/载入等功能在当前源码中未完全启用。
- 高格点数会显著增加计算和渲染时间。
- 软件主要面向可视化和教学展示，不应替代高精度量子化学计算软件。
- 当前版本主要显示实轨道和正负相位，不提供完整复波函数相位、相位色轮或复值分子轨道分析。
- 电子云显示适合教学解释概率分布，但若要进行定量统计比较，应固定采样策略、随机种子、点数和显示参数，并在文章中说明。
- 软件没有内置学生测验、学习成效统计或课堂反馈模块；若文章涉及教学效果评估，需要另行设计问卷、前后测或访谈。
- 与 Multiwfn、Gabedit、Avogadro、VMD 等通用软件相比，本软件不读取复杂量子化学输出文件，也不承担分子动力学、电子密度拓扑分析或分子轨道批处理任务。
