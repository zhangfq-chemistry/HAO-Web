## 1. 软件简介

**HAO-Web (Hydrogen Atomic Orbital Viewer)** 是一个纯前端、无后端依赖的氢原子轨道三维可视化应用。它基于量子力学的薛定谔方程解析解，能够直接在浏览器中实时计算并渲染任意量子数组合 $(n, l, m)$ 的氢原子（或类氢原子）波函数 $\Psi(r,\theta,\varphi)$。

该软件的核心技术栈包括：

- **Three.js**：WebGL 三维渲染引擎。
- **Marching Cubes**：移动立方体等值面提取算法，用于生成原子轨道的正负相位等势面。
- **Web Worker**：后台多线程计算，保证高网格密度下界面的流畅性。
- **纯 JavaScript 数学引擎**：手写拉盖尔多项式（Laguerre Polynomials）、连带勒让德多项式（Associated Legendre Polynomials）和实球谐函数（Real Spherical Harmonics），具有极高的数值精度。

软件同时集成了丰富的**课堂互动教学功能**，涵盖画板预测、轨道配对、节点发现、错误诊断、轨道对比、分步动画演示和讲解标注等，适合物理化学教师和学生使用。

## 2. 系统要求与快速启动

### 2.1 系统要求

| 项目 | 最低要求 |
|------|---------|
| 浏览器 | 支持 WebGL 的现代浏览器（Chrome 90+、Edge 90+、Firefox 90+、Safari 15+） |
| 网络 | 首次加载需联网（用于获取 Three.js 库），后续可利用浏览器缓存离线使用 |
| 硬件 | 独立显卡或集成 GPU（Intel UHD 以上），推荐 4GB+ 内存 |

### 2.2 在线使用

如果已部署至 GitHub Pages 或其他静态托管平台，直接在浏览器中访问对应 URL 即可。

### 2.3 本地运行

```powershell
# 确保已安装 Node.js（14+）
cd web
node server.mjs
```

在浏览器中打开：`http://127.0.0.1:5173`

### 2.4 部署至静态托管

将 `web/` 文件夹整体上传至 GitHub Pages、Vercel、Netlify 或 Nginx 即可。当前版本通过 CDN（`unpkg.com`）引入 Three.js。如需完全离线部署，需将 Three.js 文件下载到本地并修改 `index.html` 中的 `importmap`。

## 4. 四大核心视图详解

### 4.1 主轨道视图（Atomic Orbital）

这是软件的核心窗口，展示 3D 空间中的原子轨道。

**可展示的元素：**
- **等值面**：使用 Marching Cubes 算法在 $\Psi = +c$ 和 $\Psi = -c$ 处提取等势面。默认正瓣显示为紫色/品红色，负瓣显示为青色/蓝绿色。
- **线框 / 光滑模式**：可选择以线框网格或平滑渲染方式显示等值面。
- **电子云（Point Cloud）**：使用蒙特卡洛随机采样，按 $|\Psi|^2$ 的概率密度在空间中打点，模拟电子出现的概率分布。支持动态播放（渐进式出点）。
- **剖面切割**：XY / YZ / XZ 三种截面可将轨道切开，暴露内部结构。
- **切角模式**：1/4 角、1/8 角、3/4 角切除，用于观察轨道内部。
- **节面（Nodal Surfaces）**：可同时显示径向节面（球面）、极向节面（锥面）和经向节面（平面），以白色半透明曲面形式叠加在轨道上。
- **3D 切片投影**：在轨道旁边实时生成一个 3D 山峰图，直观展示切片位置的概率密度分布。
- **坐标轴**：X（红）、Y（绿）、Z（蓝）三轴和边界盒。
- **投影等值线**：在轨道的指定平面上叠加投影的 2D 等值线图。

**状态栏（底部）：**
- 显示当前轨道名称、正/负点数、节点信息、切片位置、盒子半径和等值面值。
- 包含 `Reset view` 按钮可一键重置相机视角。

### 4.2 投影视图（Projection）

显示轨道在特定投影平面上的 2D 概率密度热图（等值线图）和 3D 山峰图。

**展示内容：**
- **2D 等值线热图**：使用彩色填充展示波函数在投影面上的概率密度分布，正值为暖色，负值为冷色。
- **3D 山峰图**：将概率密度以三维立体山峰形式呈现，使峰值和谷值一目了然。
- **剖面切片**：一个与主视图切片联动的平面，标记当前的切片位置。
- **虚线包围盒**：标识投影区域的空间范围。

### 4.3 径向分布视图（Radial）

以三维曲面和曲线展示径向波函数的分布。

**可选显示模式（互斥）：**
- **R(r)**：径向波函数本身。
- **R²(r)**：径向概率密度。
- **RDF**：径向分布函数 $4\pi r^2 R^2(r)$，最常用。

**附加显示选项：**
- **横截面 (Cross Section)**：以圆盘截面形式展示径向函数，清晰呈现节点位置。
- **侧视图 (Side View)**：从侧面展示径向分布的 3D 曲面旋转体。
- **曲线 (Curve)**：以管状 3D 曲线展示径向函数随半径 $r$ 的变化。
- **坐标轴 (Axes)**：显示径向视图专用的坐标轴。

### 4.4 角向分布视图（Angular）

以极坐标形式在 3D 空间中展示球谐函数 $Y_l^m(\theta, \varphi)$ 的纯角向分布。

**展示内容：**
- 正/负相位的角向分布曲面，使用与主视图一致的颜色方案。
- 独立的 XYZ 坐标轴参考系。
- 支持自由旋转和缩放，独立于其他三个视图。

## 5. 控制面板（侧边栏）完整指南

点击右上角的 **⚙ (齿轮)** 按钮可展开/折叠右侧设置面板。面板分为多个可折叠的分组：

### 5.1 波函数 (Wavefunction)

| 控件 | 说明 |
|------|------|
| `n` | 主量子数（1~6），决定轨道大小和能级 |
| `l` | 角量子数（0 ~ n-1），决定轨道形状 |
| `\|m\|` | 磁量子数绝对值（0 ~ l），决定轨道空间取向 |
| `Cos型` | 选择实球谐函数的 cos 类型（如 $P_x$, $D_{xz}$ 等） |
| `Sin型` | 选择实球谐函数的 sin 类型（如 $P_y$, $D_{yz}$ 等） |
| `电子云` | 勾选后显示蒙特卡洛点云模式 |
| `▶ 播放` | 动态采样动画，逐步增加电子云打点数量 |
| 预设下拉菜单 | 快速选择常用轨道：1s, 2s, 2Pz, 2Px, 3s, 3Pz, 3dz2, 3dxz, 3dx2-y2, 4fz3 |
| `显示` 按钮 | 触发重新计算与渲染 |

> **提示**：`Cos型` 和 `Sin型` 互斥。修改量子数后需点击"显示"或从预设菜单选择来触发渲染。

### 5.2 轨道显示 (Display)

| 控件 | 说明 |
|------|------|
| `正瓣` | 显示/隐藏波函数正相位 ($\Psi > 0$) 的等值面 |
| `负瓣` | 显示/隐藏波函数负相位 ($\Psi < 0$) 的等值面 |
| `网格` | 线框模式渲染 |
| `光滑` | 光滑曲面模式渲染（与网格互斥） |
| `投影` | 在主视图的投影平面上叠加 2D 等值线 |
| `3D切片` | 在主视图中生成 3D 概率密度山峰图 |
| `xy截面` / `yz截面` / `xz截面` | 选择剖面切割平面（互斥） |
| `1/4角` / `1/8角` / `3/4角` | 切角显示模式（互斥），暴露轨道内部结构 |

### 5.3 径向 (Radial)

| 控件 | 说明 |
|------|------|
| `R` | 显示径向波函数 $R(r)$ |
| `R2` | 显示径向概率密度 $R^2(r)$ |
| `RDF` | 显示径向分布函数 $4\pi r^2 R^2(r)$（默认选中） |
| `横截` | 圆盘横截面渲染 |
| `侧视` | 3D 旋转体侧面渲染 |
| `曲线` | 管状 3D 曲线渲染（默认选中） |
| `坐标` | 径向视图坐标轴 |

> **提示**：R / R² / RDF 三者互斥，至少选择一项。横截、侧视、曲线、坐标可自由组合。

### 5.4 节面 (Nodes)

| 控件 | 说明 |
|------|------|
| `径向` | 显示径向节面（波函数为零的同心球面），节面数 = $n - l - 1$ |
| `极向` | 显示极向节面（波函数为零的圆锥面） |
| `经向` | 显示经向节面（波函数为零的半平面） |

> **教学提示**：节面的总数 = $n - 1$。其中径向节面 = $n - l - 1$，角向节面 = $l$。

### 5.5 快捷按钮面板

| 按钮 | 说明 |
|------|------|
| `课堂互动` | 打开课堂互动浮动窗口 |
| `生成动画` | 启动全屏分步演示动画 |
| `画板` | 打开手绘画板浮动窗口 |
| `讲解标注` | 进入/退出讲解标注模式 |

## 8. 菜单系统

顶部的菜单栏包含四个下拉菜单：

### 8.1 文件菜单
- **重新显示**：重新计算并渲染当前轨道。

### 8.3 帮助菜单

- **项目路径**：在状态栏显示项目路径。

### 8.4 Language 菜单
- **中文 / English**：切换界面语言。目前支持中文和部分英文标签。

## 9. 浮动窗口系统

所有浮动窗口共享统一的操作方式：

- **拖动**：按住窗口标题栏拖动可自由移动位置。
- **最小化 (－)**：点击标题栏的减号按钮，窗口收缩为标题栏。
- **最大化 (□)**：点击标题栏的方框按钮，窗口全屏化。双击标题栏也可切换最大化。
- **关闭 (×)**：点击关闭按钮隐藏窗口。

### 9.1 盒子设置窗口 (Box Settings)

精细调整计算参数的高级面板：

| 参数 | 说明 |
|------|------|
| `尺寸` | 计算空间的边长（Bohr 单位），决定渲染范围大小 |
| `显示盒子` | 是否显示虚线边界盒 |
| `格点数` | Marching Cubes 采样网格的分辨率（20~200），越大越精细但越慢 |
| `3D网格` | 在主视图中显示 3D 网格辅助线 |
| `间隔` | 网格点间距（只读，由尺寸和格点数自动计算） |
| `等值面` | Marching Cubes 的等值面阈值，控制轨道的"膨胀程度" |
| `预设值` | 将等值面和盒子大小恢复为当前轨道的推荐值 |
| `总概率` | 当前等值面所包围区域的电子出现总概率（只读） |
| `重画` | 应用新参数并重新渲染 |

### 9.2 调色板窗口 (Palette)

| 参数 | 说明 |
|------|------|
| `正瓣` 颜色 | 自定义波函数正相位的颜色 |
| `负瓣` 颜色 | 自定义波函数负相位的颜色 |
| `背景` 颜色 | 自定义所有视图的背景色 |

## 10. 课堂互动模块

点击侧边栏的 **「课堂互动」** 按钮（或底部「课堂互动」标签），将打开课堂互动浮动窗口。该窗口顶部有七个模式标签页：

### 10.1 预测-验证模式

**教学目标**：训练学生在看到真实渲染前主动思考轨道形状。

**操作流程：**
1. 从「目标轨道」下拉菜单选择一个轨道（如 2Px、3dz2 等）。
2. 在左侧画板上手绘你预测的轨道形状、节面位置和正负相位分布。
3. 点击 **「显示验证」** 按钮，右侧会渲染出真实的投影图像，同时四大视图也会切换到该轨道。
4. 对照画板预测与真实结果，分析差异。
5. 点击 **「清空内容」** 可重置画板。

### 10.2 预测评分模式

**教学目标**：量化评估学生对轨道节点的理解程度。

**操作流程：**
1. 系统会基于当前选择的目标轨道给出问题。
2. 学生填写该轨道的**径向节点数**和**角向节点数**。
3. 点击 **「提交诊断并评分」** 按钮。
4. 系统自动评分，并在下方显示参考得分、正确的节点数以及检查要点。

### 10.3 错误诊断模式

**教学目标**：帮助学生定位常见的概念错误。

提供三种诊断类型：
- **节点诊断**：检查学生是否遗漏径向节点或角向节点。自动在主视图中打开节面显示。
- **相位诊断**：提示 $P_x/P_y/P_z$、$D/F$ 轨道的方向混淆问题。
- **投影诊断**：检查学生对不同投影面上图案的理解是否正确。

点击诊断卡片，主视图会自动切换到对应的显示模式，方便即时确认。

### 10.4 节点发现模式

**教学目标**：以"先猜后看"的方式帮助学生建立节面概念。

| 按钮 | 功能 |
|------|------|
| `显示节点` | 一键打开所有节面（径向、极向、经向）的显示 |
| `隐藏节点` | 一键关闭所有节面的显示 |
| `预测节点` | 跳转到预测-验证模式，让学生先绘制预测 |

### 10.5 轨道对比模式

**教学目标**：并排比较两个不同轨道的异同。

**操作流程：**
1. 在「轨道 A」和「轨道 B」的下拉菜单中分别选择一个轨道。
2. 点击 **「开始对比」** 按钮。
3. 两个独立的 3D 渲染窗口会并排显示两个轨道，各自可以自由旋转。
4. 窗口下方自动生成分析文字，提示对比重点：
   - 若角向类型相同（如 2Px vs 3Px），重点比较径向节点的变化。
   - 若角向类型不同（如 3dz2 vs 3dxz），重点比较节面方向和相位分布。
5. 点击 **「载入主视图」** 可将对应轨道加载到主窗口进行更详细的探索。

### 10.6 轨道配对挑战模式

**教学目标**：以游戏化方式训练轨道识别能力。

**操作流程：**
1. 点击 **「随机出题」** 按钮，系统随机选择一个轨道并渲染到四大视图中，但隐藏轨道名称。

4. 点击 **「揭晓答案」** 按钮查看是否正确。
5. 系统自动记录累计答对率（如 3/5）。

## 14. 波函数公式窗口（Formula）

该窗口将当前轨道的量子力学公式拆解为各组成部分：

- **波函数总表达式**：$\Psi(r,\theta,\varphi) = R_{nl}(r) \cdot Y_l^m(\theta,\varphi)$
- **径向部分说明**：描述 $R_{nl}(r)$ 的特征，列出径向节点数。
- **角向部分说明**：描述 $Y_l^m(\theta,\varphi)$ 的轨道朝向，列出角向节点数。
- **视觉解读**：解释等值面、投影、径向分布函数和电子云在物理上的含义。
- **相位信息**：显示当前正/负相位的可见状态。

## 18. 高级功能

### 18.1 Web Worker 多线程计算

软件使用两个 Web Worker 分别计算正/负相位的标量场。当格点数设置较高（如 120+）时，计算会自动分配到后台线程，主界面保持完全流畅。

### 18.2 轨道缓存系统

软件内置了多级缓存：
- **轨道网格缓存**（最近 6 个轨道）：切换预设时可能直接命中缓存，实现即时渲染。
- **Marching Cubes 源数据缓存**（最近 4 个）：减少重复的标量场计算。

### 18.3 自动匹配桌面端参数

勾选「Desktop matched defaults」后，每次切换轨道时，盒子大小、等值面阈值和格点数会自动匹配桌面 Qt/VTK 版本的推荐参数，确保两个版本的视觉效果一致。

### 18.4 全屏模式

菜单 > 窗口 > 全屏 或使用浏览器快捷键 F11，可进入全屏沉浸式模式。屏幕上会出现「退出全屏」按钮便于回退。

## 19. 常见问题（FAQ）

**Q1：网格密度调高后计算卡顿怎么办？**

> A：高密度计算（格点数 > 120）确实需要较多计算资源。建议：
> - 实时操作时使用中等密度（60~90），展示截图时临时调高。
> - 软件已使用 Web Worker 后台计算，不会导致浏览器崩溃，耐心等待即可。

**Q2：为什么切换轨道后没有变化？**

> A：修改量子数后需要点击「显示」按钮或从预设下拉菜单中选择来触发渲染。直接修改数字框不会自动触发。

**Q3：如何保存课堂互动的结果？**

> A：使用工具栏的 ▣（导出图片）按钮，会将当前完整画面（包括浮动窗口内容）截取为 PNG。配合四宫格模式效果最佳。

**Q4：视频导出失败怎么办？**

> A：视频导出依赖浏览器的 `MediaRecorder API`。请确保使用 Chrome 或 Edge 浏览器。Safari 和部分旧版 Firefox 可能不支持 WebM 格式。

**Q5：如何让轨道更"饱满"或更"紧凑"？**

> A：打开「盒子设置」窗口，调整**等值面**参数：
> - 减小等值面数值（如 0.005）：轨道变大、更饱满。
> - 增大等值面数值（如 0.04）：轨道变小、更紧凑、只展示核心区域。

**Q6：浅色主题下颜色不够明显怎么办？**

> A：打开调色板窗口（菜单 > 窗口 > 画板 > 调色板），选择对比度更强的正/负瓣颜色。推荐在浅色主题下使用深红/深蓝配色。

**Q7：3D 切片投影的山峰图位置不对？**

> A：山峰图的位置由切片滑块控制。滑动切片控件可以实时调整山峰图和切片面的 Z 坐标位置，使其与等值面的边界精确对齐。

## 20. 教学场景最佳实践

### 场景一：课前预习

1. 学生在手机/平板上打开网页。
2. 使用「平板课堂」模式下的「大按钮模式」。
3. 从预设菜单选择目标轨道进行自主探索。
4. 使用「画板」记录自己观察到的特征。

### 场景二：课堂讲授

1. 教师将网页投屏到教室大屏。
2. 使用「生成动画」进行从电子云到轨道的分步演示。
3. 暂停在关键步骤，进入「讲解标注」模式添加注释。
4. 使用 XOY / YOZ / XOZ 按钮快速切换视角，讲解轨道对称性。
5. 打开「公式窗口」，将数学表达式与 3D 视觉对应。

### 场景三：课堂互动

1. 使用「预测-验证」模式：先让学生画出预测，再揭晓真实结果。
2. 使用「轨道配对挑战」模式：随机出题，学生抢答。
3. 使用「轨道对比」模式：将 2Px 与 3Px 并排对比，讲解 n 增大时径向节点的增加。
4. 使用「节点发现」模式：先隐藏节面让学生猜测，再一键显示。

### 场景四：作业与论文截图

1. 调整到目标轨道和视角。
2. 切换到「四宫格」布局。
3. 按住 Shift 键点击导出按钮，获取 8192 像素的超高清四宫格截图。
4. 导出图片已自带标题标注，可直接用于报告和论文。
