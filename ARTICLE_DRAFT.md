# HydrogenAtomOrbitalViewer：面向探究式学习的氢原子轨道三维可视化工具

## 摘要

氢原子轨道是普通化学、结构化学和物理化学教学中的核心概念，但学生常常难以从公式或二维教材插图中建立稳定的三维空间图像。本文报道一个基于 Qt 和 VTK 开发的桌面端氢原子轨道可视化工具 HydrogenAtomOrbitalViewer，用于支持探究式学习活动和课堂中的表征能力训练。该工具以“可操控的轨道虚拟实验室”为设计目标，将量子数选择、实轨道表示、三维等值面、电子云、角向函数、径向函数、轨道投影、径向/极向/经向节面和裁切视图整合到同一工作流中。学生可以通过改变 `n`、`l`、`m`、轨道类型、显示模式和截面方式收集可视化证据，进而归纳量子数、节点、相位和径向分布之间的关系。与通用分子可视化或波函数分析软件相比，该工具不追求复杂电子结构后处理，而是围绕本科教学中最常见的类氢原子轨道概念提供低门槛、可交互、可导出图片的本地平台。本文介绍软件的理论基础、功能设计、图像表示策略和示例图组，并提出面向课堂实施的探究式活动、检索练习和学习评价方案。其主要亮点在于多窗口联动、多表示互补、节点显式化、径向-角向联合解释以及高质量教学图像导出。

氢原子轨道；探究式学习；三维可视化；Qt；VTK；表征能力；计算机辅助教学

## 引言

原子轨道贯穿本科化学课程体系。学生在普通化学中首先接触主量子数 `n`、角量子数 `l` 和磁量子数 `m`，并学习将这些量子数与 `s`、`p`、`d` 等轨道形状联系起来；在物理化学和量子化学中，学生进一步学习氢原子薛定谔方程、径向函数、球谐函数、概率密度和节面。尽管轨道概念十分重要，学生对轨道的空间形状、相位和节点结构仍常存在理解困难。这一问题不仅影响氢原子轨道学习，也会延伸到分子轨道理论、成键方向和轨道重叠等后续内容。

探究式学习强调学生通过观察、操控变量、收集数据、分析证据和建立模型来获得概念理解，而不是被动接受结论。Li 等报道的 Online Orbital Explorer 和 BingOrbital Game 将原子轨道可视化工具与探究式活动、数据收集和检索练习结合起来，体现了化学教育工具从“展示图像”向“支持学习过程”转变的趋势[12,13]。对于氢原子轨道教学而言，合适的软件不应只是呈现一幅三维图，而应成为学生提出问题、改变参数、比较结果并形成解释的脚手架。

化学教育文献中长期关注轨道图像的呈现方式。早期工作讨论了原子和分子轨道模型、轨道图绘制和等值面表示方法[1-4]。随着计算机图形技术的发展，研究者提出了基于 Mathematica、Gnuplot、Winplot、Jmol 和网页工具的轨道绘图方法[5-9]，并进一步发展出三维打印模型、探究式在线活动、虚拟现实和增强现实应用[10-15]。这些研究共同说明，轨道可视化不只是美化教材插图，而是连接量子力学数学形式与化学直觉的重要桥梁。

近年来的教学工具更加重视“多表示形式”的配合。Tully 等人的点描式可视化使用概率点云、径向分布、截面、轮廓图、边界面和节面帮助学生比较同一轨道的不同图像表达[9]。Manae 和 Hazra 指出，学生容易把轨道形状简单归因于角向函数，而忽略径向函数和角向函数的共同作用，因此应在图示中同时体现径向和角向贡献[11]。Hanson 近期报道的氢原子轨道可视化 GUI 工具则强调了在本科物理化学中通过交互界面呈现三维体密度、节面和任意量子数轨道的重要性[16]。Rachek 和 Wilson 进一步把注意力集中到类氢原子的径向分布函数，通过 R Shiny 应用动态显示 `Rnl(r)`、`Rnl(r)^2` 和归一化径向分布函数，并用积分阴影帮助学生理解给定半径范围内的电子概率[25]。这些工作提示我们，面向教学的轨道工具不应只提供一张漂亮的边界面图，而应帮助学生在公式、三维形状、概率分布和节点结构之间来回转换。

现有软件如 VMD、Avogadro、Gabedit、Multiwfn 等可用于分子结构、分子轨道、电子密度和波函数分析[17-20]，但这些通用计算化学软件通常面向更宽泛的研究场景。对教学而言，一个专门面向氢原子轨道的轻量化交互工具具有独特价值：它可以将量子数、轨道名称、节面类型、径向函数和相位瓣集中在同一界面中，降低学生探索轨道形状的门槛。与此同时，增强现实和虚拟现实研究表明，三维空间交互对分子几何、对称性和轨道理解具有潜在帮助[14,15,26,27]；桌面工具虽然沉浸感较弱，但具有部署简单、运行稳定、便于课堂投影和导出图像等优势。

本文基于 HydrogenAtomOrbitalViewer 项目和使用手册，介绍一个使用 Qt Widgets 构建界面、使用 VTK 完成三维渲染的轨道可视化程序。与 Online Orbital Explorer 采用 Web 工具支持探究式学习的思路类似[13]，本文将 HydrogenAtomOrbitalViewer 定位为本地桌面端的轨道虚拟实验室：教师可用它组织课堂观察和检索练习，学生可用它收集截图、比较轨道、解释节点并生成学习报告。本文首先概述氢原子轨道的理论基础和实轨道表示，然后介绍软件实现与界面功能，随后结合实际截图说明其多表示图像策略，最后讨论探究式活动设计、亮点、局限性和后续改进方向。

## 理论基础：氢原子轨道

氢原子轨道是氢原子定态薛定谔方程的波函数解。在球坐标中，氢原子波函数通常可分解为径向部分和角向部分的乘积：

```text
Psi(n,l,m;r,theta,phi) = R(n,l;r) Y(l,m;theta,phi)
```

其中 `n` 为主量子数，`l` 为角量子数，`m` 为磁量子数。`R(n,l;r)` 为径向函数，决定电子分布随核间距离 `r` 的变化；`Y(l,m;theta,phi)` 为球谐函数，决定轨道的角向形状。量子数满足：

```text
n = 1, 2, 3, ...
l = 0, 1, ..., n-1
m = -l, ..., 0, ..., l
```

在化学教学中，`l=0,1,2,3,4` 分别对应 `S/P/D/F/G` 轨道。轨道概率密度与波函数模平方相关：

```text
rho = |Psi|^2
```

因此，轨道可视化可以选择显示波函数等值面、概率密度等值面或空间点云。不同表示方式强调的物理含义不同：等值面有利于观察轨道外形和相位，电子云或点云有利于体现电子概率分布的空间疏密，径向分布函数则更适合说明电子在不同半径附近出现的概率。

HydrogenAtomOrbitalViewer 在界面中同时提供原子轨道三维视图、角向函数视图、径向函数视图和轨道投影视图。这样的多视图设计有助于学生把波函数的数学分解与图形表现对应起来：径向视图对应 `R(r)`、`R^2(r)` 和径向分布函数，角向视图对应球谐函数的空间形状，原子轨道视图则综合显示完整轨道。

在教学解释中需要特别强调，角向函数并不能单独决定完整轨道图像。角向函数控制方向性和角向节点，径向函数控制轨道随半径变化的振荡、径向节点和概率分布范围。例如 `2Pz` 和 `3Pz` 都具有 p 型角向特征，但 `3Pz` 由于 `n-l-1=1` 而具有一个径向节点。因此，同样的 p 型方向性可以对应不同的径向结构。软件将角向函数、径向函数和完整轨道并列显示，正是为了帮助学生避免把“轨道形状”简化为单一角向图像。

节点数也可作为连接量子数和图像的一条清晰主线。氢原子轨道的角向节点数为 `l`，径向节点数为 `n-l-1`，总节点数为 `n-1`。在软件中，用户可通过径向、极向和经向节面选项观察这些节点，并借助 `xy`、`xz`、`yz` 截面或空间裁切查看内部结构。将节点公式与三维图像对应起来，有助于学生把抽象的量子数规则转化为空间图像判断。

## 实轨道表示与相位显示

复球谐函数中含有与方位角有关的复指数项。对于 `m != 0` 的轨道，单独使用复球谐函数时会出现实部、虚部和模值形状不完全符合普通化学中 `Px/Py` 等轨道图像直觉的问题。Hanson 对这一问题进行了详细说明，并采用线性组合构造实轨道，使轨道形状与本科化学中常用的实轨道表示相联系[16]。

本项目也采用化学教学中常见的实轨道命名和显示方式。例如：

- `Pz` 对应 `m = 0`。
- `Px` 和 `Py` 可视为 `m = ±1` 角向函数的实组合。
- `Dxy`、`Dxz`、`Dyz`、`Dx2-y2`、`Dz2` 用于表示五个常见 d 轨道方向。

软件界面中的 `Cos型` 与 `Sin型` 用于区分 `m != 0` 时的两类实球谐组合。三维视图中，正瓣和负瓣使用不同颜色显示，便于区分波函数相位。对于文章或教学展示而言，正负相位的可视化有助于解释轨道对称性、节点结构和后续成键/反键概念中的相位匹配问题。

需要说明的是，本软件的相位显示主要服务于本科化学中常见的实轨道图像。对于复值分子轨道或完整复波函数，相位可视化需要额外的颜色映射或相位色轮。Al-Saadon 等人关于复值分子轨道的工作表明，实部、虚部、振幅和相位的表示会影响学习者对分子轨道图像的理解[23]。因此，本文将本软件定位为实轨道和教学相位显示工具，而不是复值波函数分析平台。

## 设计目标与可视化策略

基于上述文献和手册整理，HydrogenAtomOrbitalViewer 的设计目标可概括为三点。第一，降低使用门槛。用户可以通过量子数或轨道名称选择目标轨道，不需要编写绘图脚本或准备量子化学输出文件。第二，突出多表示形式。软件同时提供完整轨道、角向函数、径向函数、投影、电子云和节面显示，使同一数学对象可从多个角度观察。第三，服务教学图像生成。程序允许调节等值面、盒子尺寸、格点数、颜色、透明度和视角，并可导出 PNG 图片。

在图像表示策略上，本文建议把软件输出分为四类。等值面图用于展示轨道边界、对称性和相位，但不应被解释为电子的硬边界。电子云图以点分布表达概率密度，适合说明电子出现的空间疏密。截面和裁切图用于观察轨道内部节点，能够弥补单纯外表面图隐藏内部结构的不足。径向函数和径向分布函数用于把空间图像与数学函数对应起来，特别适合解释径向节点和电子最可能出现的半径范围。这样的分类与 Tully 等人的多表示思路、Manae 和 Hazra 的径向-角向联合图示以及 Rachek 和 Wilson 的径向分布工具形成呼应[9,11,25]。

## 工具亮点与教学创新

参照 Online Orbital Explorer 的工具报告写法[13]，HydrogenAtomOrbitalViewer 的教育价值不仅在于“能画出轨道”，还在于它为课堂探究提供了可操作的变量、可观察的现象和可保存的证据。其主要亮点如下。

**亮点一：多窗口联动降低认知分离。** 传统教学中，完整轨道图、径向函数图、角向函数图和投影图往往分散在教材不同位置。学生需要在多个静态图之间切换，容易把这些图理解为互不相关的对象。本软件在同一界面中并列显示总波函数、角向函数、径向函数和投影，使学生能够把“公式分解”和“图像分解”对应起来。

**亮点二：显式化节点，支持基于证据的推理。** 许多学生会记住“节点数等于 `n-1`”之类规则，却难以在三维图像中指出节点的位置。本软件将径向、极向和经向节面作为可开关的可视对象，并支持截面和裁切显示，使学生可以从图像中直接观察节点，而不是只依赖教师口头描述。

**亮点三：同时呈现等值面和电子云，减少实体化误解。** 轨道等值面容易被初学者误解为电子运动的硬边界。软件提供电子云点分布和等值面之间的切换，使学生能够比较“边界面图”和“概率点云图”的差别，从而理解轨道本质上是概率分布的表征。

**亮点四：可导出高质量图像，服务学习报告和论文写作。** 软件可将当前活动窗口导出为 PNG 图片，并允许调节颜色、透明度、格点数、盒子尺寸和显示方式。学生可以把截图作为探究活动的数据证据，教师也可以生成风格统一的课堂材料和文章插图。

**亮点五：本地桌面程序适合离线教学环境。** 与 Web 工具相比，本软件不依赖浏览器环境、网络连接或服务器部署；与通用研究软件相比，本软件界面更聚焦于氢原子轨道教学。对于需要离线演示、固定机房教学或稳定课堂投影的场景，桌面端 Qt/VTK 架构具有现实优势。

## 软件设计与实现

### 开发环境与总体结构

HydrogenAtomOrbitalViewer 是一个桌面 GUI 程序，当前项目目标名为 `HAO`。程序使用 Qt 构建窗口、菜单、工具栏、选项卡和参数面板，使用 VTK 负责三维等值面、节点、坐标轴、盒子和电子云的渲染。根据当前 `src.pro`，项目使用 Qt 6.10.1，依赖 `core`、`gui`、`widgets`、`charts`、`datavisualization` 和 `concurrent` 模块，三维渲染依赖 VTK 9.6。

项目的主要模块如下：

- `AO.cpp/.h`：原子轨道核心数据模型，负责轨道名称与量子数转换、波函数计算、体数据生成、等值面构建、电子云点生成和节面构建。
- `functionals.cpp/.h`：数学函数库，包含阶乘、Laguerre 多项式、Legendre 多项式、径向函数、角向函数、概率函数和 Lambert W 相关函数。
- `View3D.cpp/.h`：原子轨道三维显示模块。
- `Angular3D.cpp/.h`：角向函数三维显示模块。
- `Radial3D.cpp/.h`：径向函数三维显示模块。
- `Proj3D.cpp/.h`：轨道投影显示模块。
- `orbitalProperty.cpp/.h`、`axisProperty.cpp/.h`：颜色、透明度、线宽、背景和坐标轴设置对话框。

图 1 展示软件启动后的主界面，包括左侧控制面板和右侧多窗口可视化区域。图中以 `4Dx2-y2` 为例，同时显示波函数投影、总波函数等值面、径向分布函数和角向函数轮廓图。这个界面体现了软件的核心设计：使用者不需要在多个程序之间切换，就可以把完整轨道、角向部分、径向部分和投影图像放在同一屏幕中比较。

### 氢原子轨道显示功能

在氢原子轨道页签中，用户可以通过两种方式选择轨道：一是直接设置 `n`、`l` 和 `m`，二是从轨道名称下拉框中选择 `1S`、`2Px`、`3Dz2`、`5Gxyz2` 等轨道。点击 `显示` 后，程序根据当前轨道名称解析量子数，生成波函数数据并刷新四个视图。

程序支持以下显示方式：

- 原子轨道等值面：显示完整轨道形状。
- 网格面与光滑面：分别用于展示表面结构和生成更美观的教学图像。
- 正瓣/负瓣控制：显示或隐藏不同相位区域。
- 电子云：以空间点分布显示电子概率密度。
- 节面：显示径向、极向和经向节面。
- 截面与裁切：显示 `xy`、`xz`、`yz` 截面以及 `1/8`、`1/4`、`3/4` 空间裁切。
- 投影：辅助观察轨道在不同平面上的投影关系。

图 2 以 `3Dyz` 为例展示电子云显示模式。与光滑等值面相比，电子云使用大量离散点表示电子概率密度的空间疏密，更容易让学生意识到轨道并不是具有硬边界的实体。图中还保留了角向函数和投影窗口，可用于说明同一轨道的点云、投影和角向轮廓之间的对应关系。

### 径向函数与节点显示

软件在径向视图中提供 `R`、`R2` 和 `RDF` 三种显示模式。`R` 可表现径向波函数的符号变化，`R2` 表现径向函数平方，`RDF` 更适合说明电子在不同半径处的分布趋势。通过这些视图，学生可以把三维轨道中的径向节点与函数图像中的零点相联系。

图 3 以 `3S` 为例展示径向节点和空间裁切。s 轨道角向部分具有球形对称性，但当主量子数增大时，径向函数会出现节点，使完整轨道内部形成同心壳层结构。该图通过 `3/4` 裁切和径向分布窗口把内部节点暴露出来，适合用于解释“s 轨道并不总是简单实心球体”。

## 探究式学习活动与课堂实施建议

该软件可用于课堂演示、学生自主探索和课程作业。根据已有文献，三维可视化工具可帮助学生将抽象量子数与具体空间形状联系起来，也可改善学生对空间取向、分子轨道和电子分布的理解[12-16,21,22]。借鉴 Online Orbital Explorer 和 BingOrbital Game 的思路[13]，HydrogenAtomOrbitalViewer 可被设计为探究式学习中的“数据收集工具”和课后检索练习的“图像题库生成器”。

### 探究活动 1：从量子数预测轨道形状

该活动面向普通化学或结构化学中第一次学习轨道的学生。教师给出一组轨道名称，例如 `1S`、`2Pz`、`3Dxy` 和 `4F`，要求学生先根据 `n`、`l`、`m` 预测轨道的大致形状、角向节点数和径向节点数，再用软件生成图像进行验证。学生需要记录预测、截图证据和修正后的解释。该活动的重点不是让学生记住某一张图，而是训练他们用量子数规则解释可视化结果。

### 探究活动 2：径向函数是否会改变轨道图像

该活动针对“轨道形状只由角向函数决定”的常见误解。学生比较 `2Pz` 和 `3Pz`，同时导出完整轨道、角向函数、径向函数和 RDF 图像。教师可引导学生回答三个问题：两者角向函数有什么共同点？径向函数有什么不同？这些不同如何反映在完整轨道图像中？该活动直接回应 Manae 和 Hazra 所强调的径向-角向联合理解问题[11]。

### 探究活动 3：轨道是边界面还是概率分布

该活动要求学生对同一个轨道分别显示光滑等值面、电子云、截面和 RDF，并比较四种图像的表达对象。学生应写出每种表示能回答的问题，例如“哪里是波函数正负相位区域”“哪里概率密度更高”“内部节点在哪里”“电子在哪个半径范围附近更可能出现”。该活动可帮助学生避免把等值面误解为电子运动边界。

### 探究活动 4：节点识别和图像检索练习

借鉴 BingOrbital Game 的检索练习思想[13]，教师可用本软件提前导出若干轨道图像，去掉轨道名称后制作“轨道识别卡”。课堂上学生根据图像中的瓣数、相位、节点和径向结构判断轨道名称，并说明判断依据。与单纯选择题相比，这种活动要求学生从图像中提取证据，属于更高层次的表征能力训练。

### 学习评价建议

如果需要把本文发展成包含教学评估的研究，可在这些活动基础上增加前测和后测。前测可考查学生是否能识别轨道形状、节点和相位；后测可要求学生解释 `2Pz` 与 `3Pz`、等值面与电子云、`3Dx2-y2` 与 `4Dx2-y2` 的差异。也可加入图形解释题、截图标注题和简短访谈，以评价学生是否真正建立了径向-角向联合理解。评价指标可包括：节点识别准确率、图像解释质量、能否用量子数预测图像特征、能否区分波函数相位和概率密度。

## 与已有工作的比较

与 Winplot、Gnuplot 和 Jmol 等工具相比，本项目的特点是将氢原子轨道、径向函数、角向函数和投影集中在同一 Qt 桌面界面中，用户不需要编写绘图脚本或打开网页环境即可进行交互操作[5-9]。脚本化工具的优势在于可重复、可批量生成图像，网页工具的优势在于部署方便、跨平台访问；本项目的优势则在于本地运行、窗口响应直接、参数面板集中且便于课堂投影和导出图片。

与 Online Orbital Explorer 相比，本软件同样强调学生通过操控变量和观察图像来学习原子轨道，但两者的技术路线和教学侧重点不同[13]。Online Orbital Explorer 的优势在于基于浏览器、便于跨设备访问，并加入 RDF comparison 和 BingOrbital 游戏以支持数据比较和检索练习。本软件的优势在于本地 Qt/VTK 渲染、多子窗口联动、截面/裁切/节点显示集中，以及可直接生成高分辨率 PNG 图像，适合课堂演示、课程论文和教学文章配图。换言之，前者更像在线探究平台，后者更像可离线运行的轨道虚拟实验室。

与 Hanson 的 Python GUI 工具类似，本项目强调学生通过交互控制理解轨道三维结构和节点[16]。不同之处在于，本项目采用 Qt/VTK 桌面程序架构，并通过多种 VTK 子窗口同时提供原子轨道、角向函数、径向函数和投影的并列显示，适合在结构化学与物理化学交叉教学中使用。与 Rachek 和 Wilson 的 R Shiny 径向分布工具相比，本项目对 RDF 的数学分析深度较低，但能把径向函数直接放入三维轨道工作流中，从而更适合解释径向函数如何影响完整轨道图像[25]。

与通用可视化软件如 Avogadro、Gabedit、VMD 和 Multiwfn 相比，本工具的功能范围更窄，但更聚焦于轨道教学展示[17-20]。Avogadro、Gabedit 和 VMD 适合分子结构建模、分子显示和计算化学数据处理，Multiwfn 则适合波函数分析和多种实空间函数处理。本软件不尝试替代这些研究型工具，而是把教学常用的氢原子轨道、实球谐函数、径向图像和电子云做成更直接的演示界面。

与 VR/AR 和移动端工具相比，本项目不提供沉浸式空间体验，但对硬件要求更低，也更容易与现有课堂投影、实验报告和教学文章配图结合。Dai 等、Levy 等以及 Zambri 和 De Backere 的工作表明，沉浸式或移动增强现实能够增强学生对三维几何、对称性和轨道的空间感知[14,15,26]。本项目可看作这些沉浸式方案之前的低门槛桌面层：先帮助学生在普通屏幕上理解图像和参数，再进一步扩展到 AR/VR 或实体模型。

## 局限性、可推广性与后续改进

当前项目仍存在若干限制。首先，构建配置中包含本机绝对路径，跨机器编译前需要修改 `src.pro`。其次，原始 `readme` 要求修改 VTK `vtkActor` 并重新编译 VTK，这增加了部署难度。第三，部分 UI 文本存在历史编码问题，部分功能如 `lambertW`、径向 `All` 选项和属性保存/载入尚未完全启用。第四，较高网格点数会显著增加计算和渲染时间。

从科学表示角度看，当前版本主要显示实轨道和正负相位，不提供完整复波函数相位、相位色轮或复值分子轨道分析。电子云显示适合教学解释概率分布，但若要进行定量比较，需要固定采样策略、点数和显示参数。软件目前也不读取复杂量子化学输出文件，因此不适合承担分子轨道批处理、电子密度拓扑分析或高精度计算结果后处理任务。

从教学研究角度看，本文目前更接近工具介绍和教学设计初稿，还没有包含真实课堂实施数据。若后续面向化学教育期刊投稿，需要进一步收集学生使用反馈、前后测结果、图形解释题表现或访谈资料，以证明该工具对学习效果的影响。

尽管如此，本工具已经具备较好的可推广性。首先，软件主题聚焦，教师不需要在复杂分子建模功能中筛选适合课堂的模块。其次，图片导出功能可支持跨课程复用，教师可以预先生成轨道识别练习、课堂讲义和实验报告模板。第三，桌面端运行方式适合网络条件受限的教学场景。后续若加入预设任务、参数记录和自动图注功能，软件可进一步从“可视化工具”发展为完整的探究式学习平台。

后续工作可从以下方面改进：

- 使用 CMake 或更通用的 qmake 配置替代硬编码路径。
- 消除对修改 VTK 源码的依赖。
- 修复 UI 编码并完善中英文翻译。
- 增加预设教学活动模式，包括任务提示、参数记录、截图导出和图注生成。
- 提供预设图像导出模板，便于生成统一风格的教学插图和文章图组。
- 增加学生学习效果评估模块或配套练习，如概念题、截图标注题和前后测问卷。
- 增加复轨道或相位色轮显示模式，扩展到更高阶波函数可视化教学。
- 在保持教学聚焦的前提下，逐步支持分子轨道或多电子原子轨道示意。

## 结论

HydrogenAtomOrbitalViewer 将氢原子轨道的数学表达、三维等值面、电子云、节面、径向函数和轨道投影整合在一个交互式桌面程序中。该工具的核心价值在于多表示形式的并列呈现：学生可以在同一操作流程中比较完整轨道、角向函数、径向函数、概率点云和截面，从而把量子数、节点、相位和空间取向联系起来。仿照探究式轨道学习工具的教学理念，本文将该软件定位为可离线运行的轨道虚拟实验室。它既能作为教师课堂演示工具，也能作为学生收集图像证据、完成探究报告和进行轨道识别练习的平台。与通用计算化学软件相比，本软件功能更聚焦；与脚本或网页工具相比，本软件更强调本地桌面交互、多窗口联动和教学图片导出。尽管当前版本仍需在构建配置、编码、部署、复轨道表示和教学评估方面继续完善，它已经为本科化学课程中的氢原子轨道可视化、课堂演示和文章配图提供了一个具有推广潜力的平台。
